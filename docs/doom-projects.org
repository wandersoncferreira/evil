#+title: Doom Projects

Doom relies on [[https://github.com/bbatsov/projectile][Projectile]] to handle all project related functionalities
under =SPC p= prefix.

| Command   | Description                                    |
|-----------+------------------------------------------------|
| spc p + e | edit .dir-locals file from project             |
| spc p + i | invalidate project cache                       |
| spc p + t | toggle between project implementation and test |
